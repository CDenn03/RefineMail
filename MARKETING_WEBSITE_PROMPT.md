# Marketing Website Design Prompt for AI Email Refiner

## Project Overview
Design a modern, conversion-focused marketing website for "AI Email Refiner" - a Chrome extension that uses AI to rewrite Gmail and Zimbra emails in multiple styles (formal, friendly, concise, persuasive, apologetic, casual, or neutral/professional).

## Product Details

### Core Features
- **AI-Powered Email Rewriting**: Instantly transform email drafts using OpenAI technology
- **7 Writing Styles**: Formal, Friendly, Concise, Persuasive, Apologetic, Casual, Neutral/Professional
- **Gmail & Zimbra Support**: Works seamlessly with both email platforms
- **Custom Instructions**: Add personalized instructions for tailored results
- **Real-Time Preview**: Review changes before applying them to your email
- **One-Click Application**: Apply improved text directly to your compose window
- **Character Validation**: Ensures emails stay within optimal length (max 3000 characters)
- **Subject Line Support**: Maintains and improves email subjects

### Target Audience
- Business professionals who write frequent emails
- Sales and marketing teams needing persuasive communication
- Customer support representatives requiring empathetic responses
- Executives and managers who need formal correspondence
- Remote workers managing cross-cultural communication
- Anyone wanting to save time and improve email quality

### Key Benefits
- Save 10+ hours per week on email writing
- Improve response rates with better-crafted messages
- Maintain consistent professional tone across all communications
- Reduce email anxiety and writer's block
- Adapt communication style to different recipients instantly
- Enhance clarity and professionalism without effort

## Design Guidelines

### Visual Style
- **Modern & Clean**: Minimalist design with plenty of white space
- **Professional Yet Approachable**: Balance between corporate and friendly

### Color Theme

**Primary Colors**:
- **Navy Primary**: `#0f2a44` - Main brand color (buttons, headers, key elements)
- **Navy Secondary**: `#124170` - Darker variant for text and accents
- **Navy Hover**: `#1a5a8a` - Interactive states and hover effects

**Accent Colors**:
- **Success Green**: `#28a745` - Success messages, positive indicators
- **Alert Red**: `#e74c3c` - Error states, warnings
- **Info Blue**: `#3498db` - Informational elements, links

**Neutral Colors**:
- **White**: `#ffffff` - Primary background
- **Light Gray**: `#f9f9f9` - Secondary backgrounds, disabled states
- **Medium Gray**: `#f5f5f5` - Card backgrounds, subtle sections
- **Border Gray**: `#ddd` - Borders, dividers
- **Text Gray**: `#666` - Secondary text, captions
- **Dark Text**: `#333` - Primary body text
- **Black**: `#000000` - Headings, emphasis

**Gradient Options** (for hero and accent sections):
- **Primary Gradient**: `linear-gradient(135deg, #0f2a44 0%, #124170 50%, #1a5a8a 100%)`
- **Subtle Gradient**: `linear-gradient(180deg, #ffffff 0%, #f9f9f9 100%)`
- **Overlay Gradient**: `linear-gradient(135deg, rgba(15, 42, 68, 0.95) 0%, rgba(18, 65, 112, 0.9) 100%)`

**Shadow System**:
- **Soft Shadow**: `0 4px 14px rgba(15, 42, 68, 0.18)` - Cards, buttons
- **Medium Shadow**: `0 6px 20px rgba(15, 42, 68, 0.28)` - Hover states
- **Strong Shadow**: `0 8px 32px rgba(0, 0, 0, 0.3)` - Modals, elevated elements
- **Subtle Shadow**: `0 2px 8px rgba(0, 0, 0, 0.1)` - Minimal elevation

### Typography System

**Font Families**:
- **Primary Font**: `'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', 'Oxygen', 'Ubuntu', 'Cantarell', sans-serif`
  - Modern, highly readable, excellent for UI
  - Use for headings, body text, and UI elements
  
- **Alternative Font**: `'Poppins', 'Inter', sans-serif`
  - Slightly more playful, great for marketing
  - Use for hero headlines and feature titles
  
- **Monospace Font**: `'Fira Code', 'Courier New', monospace`
  - For code snippets and email examples
  - Maintains readability in technical content

- **Hero Headline**: `56px` / `3.5rem` - Bold, line-height: 1.1
- **H1**: `48px` / `3rem` - Bold, line-height: 1.2

- **Body Regular**: `16px` / `1rem` - Regular, line-height: 1.6
- **Body Small**: `14px` / `0.875rem` - Regular, line-height: 1.5
- **Caption**: `13px` / `0.8125rem` - Regular, line-height: 1.4

**Font Sizes** (Mobile - scale down by 20-30%):
- **Hero Headline**: `36px` / `2.25rem`
- **H1**: `32px` / `2rem`
- **H2**: `28px` / `1.75rem`
- **H3**: `24px` / `1.5rem`
- **H4**: `20px` / `1.25rem`

**Font Weights**:
- **Regular**: `400` - Body text, descriptions
- **Medium**: `500` - Emphasized text, labels
- **Semi-Bold**: `600` - Subheadings, important text
- **Bold**: `700` - Headings, CTAs, strong emphasis

**Letter Spacing**:
- **Headings**: `-0.02em` (tighter for large text)
- **Body**: `0` (default)
- **Buttons/CTAs**: `0.02em` (slightly wider for readability)
- **All Caps**: `0.05em` (wider for uppercase text)

### UI Elements
- **Buttons**: Rounded corners, clear hover states, shadow effects
- **Cards**: Subtle shadows, rounded corners, hover animations
- **Icons**: Line icons or filled icons (consistent style throughout)
- **Screenshots**: Device mockups (browser windows, Chrome extension interface)
- **Animations**: Subtle micro-interactions, smooth transitions

### Responsive Design
- Mobile-first approach
- Breakpoints: Mobile (< 768px), Tablet (768-1024px), Desktop (> 1024px)
- Ensure CTAs are easily tappable on mobile
- Stack sections vertically on mobile
- Optimize images for different screen sizes

### Performance
- Lazy load images below the fold
- Optimize all images (WebP format)
- Minimize JavaScript for fast load times
- Use CSS animations over JavaScript where possible
